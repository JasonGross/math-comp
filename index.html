About

Welcome to the GitHub pages of the Mathematical Components libraries! These Coq libraries have started as a subset of a formal proof of the Four Colour Theorem and have been used since for other large scale projects, like a formal proof of the Odd Order Theorem. They are written using the Ssreflect tactic language, which is shipped with the distribution of the libraries.

Releases

We are proud to announce the immediate availability of the Ssreflect proof language and the Mathematical Components library version 1.6 for Coq 8.4pl6 and Coq 8.5beta3. Older versions of the libraries are available from here.

Documentation

The Ssreflect language has a reference manual. The source files of the libraries each feature a documentation header which describe the concepts and notations they introduce. This page displays a coqdoc-generated html version of these sources.

Authors and Contributors

The Mathematical Components libraries and the Ssreflect tactic language are developped by the Mathematical Components team, at the Inria -- Microsoft Research Joint Centre.
